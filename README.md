# 🔢 NumPy (Numerical Python)

> Covered in `13july2026numpy.ipynb`

NumPy is a Python library for working with arrays. It is widely used in data science, linear algebra, Fourier transforms, and matrix operations. NumPy arrays (`ndarray`) are up to 50x faster than traditional Python lists due to contiguous memory storage (locality of reference).

## Getting Started

* Install: `pip install numpy` or `python -m pip install numpy`
* Import: `import numpy as np`
* Check Version: `np.__version__`

## Array Basics

* Creating Arrays (`np.array()`)
* `np.zeros()` — Pre-filled Placeholder Arrays
* `np.arange()` — Array Version of `range()`
* `np.linspace()` — Evenly Spaced Points
* 1D, 2D, and 3D Arrays
* Higher Dimension Arrays (`ndmin` parameter)
* Array Attributes (`.shape`, `.ndim`, `.dtype`)
* Type Casting (`.astype()`)

## Array Operations

* Indexing & Accessing Elements
* Slicing (`start:end`, `start:end:step`, negative indexing)
* Reverse Array (`[::-1]`)
* Copy vs View (`copy()`, `view()`, `.base` attribute)
* Shape & Reshape (`reshape()`, `flatten()`)

## Array Manipulation

* Iterating Arrays (`for`, `np.nditer()`, `np.ndenumerate()`)
* Joining Arrays (`concatenate()`, `stack()`, `hstack()`, `vstack()`, `dstack()`)
* Splitting Arrays (`array_split()`, axis parameter)
* Sorting Arrays (`np.sort()`)
* Filtering Arrays (Boolean Indexing, Manual Filtering, Even/Odd Filtering)

## Why NumPy?

* Faster than Python lists for numerical computation
* Optimized for modern CPU architectures
* Essential for data science workflows
* Supports linear algebra, matrices, and scientific computing
